## computer (hardware)
i have a couple of different computers!! mostly because i keep having computers break in ways that make them not suitable for daily use anymore, but they're not totally dead, so i refuse to get rid of them...

### desktop (Prometheus)
- basically my "main computer" and daily driver
- light gaming rig built by a friend of a friend in like 2022, using an ASRock motherboard
Windows 11 (born with Windows 10)
- AMD Ryzen 7 3800X 8-Core Processor (3.90 GHz)
- 16GB RAM (hoping to upgrade soon though...)
- NVIDIA GeForce GTX 1660 SUPER (6 GB) GPU
- born with 256GB NVMe primary drive; recently added a cheap used 1TB HDD
- planning to add an additional drive with a Linux partition eventually...

### portable computer (Valencia)
- <a href="http://www.orangepi.org/html/hardWare/computerAndMicrocontrollers/details/orange-pi-800.html" rel="noopener noreferrer">Orange Pi 800</a> keyboard computer: a single-board computer (yknow, like a Raspberry Pi) but with a keyboard as its case
- bought this so i can bring it when i go to my makerspace!! (way cheaper than a laptop lol) unfortunately it doesn't have a monitor so i can't bring it many other places; makerspace only works because there's already monitors that live there
- Ubuntu (Orange Pi's customized variant of Xubuntu)
- ARM64
- Rockchip RK3399 SoC
- 4GB RAM
- 64GB eMMC storage
- handmade matching fabric case c:
- believe it or not, this little buddy can do *pretty much* everything i expect from a PC!!! the only real shortcoming is the lack of OpenGL support.

### monitor
- NEC MultiSync
- 1280x1024
- it's an old VGA monitor BUT i like it for two reasons!!
    - the "multisync" thing: you can connect two VGA sources at the same time and toggle between them! really handy for working with my orange pi!
    - i actually really like that it has a less common resolution that's more square-ish than usual monitors these days!! i do a whole lot of writing, and a taller screen is so much nicer for that! the only time it sucks is that video games sometimes struggle to match the resolution...

### laptop (Tegu)
- Toshiba Satellite L755D
- Xubuntu partition + Windows 10 partition that i gotta delete
- it's like 12 years old or smth lol
- i thought it was kinda dead but then i tried it recently and it turned out it actually works again???

## software

### Obsidian
- markdown-based note taking app ([Obsidian website](https://obsidian.md/))
- closed source but free

my true daily driver. i journal in here almost every day (it holds like 2 years of journal entries now i think!), plus use it to track all my projects, and basically anything i want to remember later.

#### Obsidian plugins
- Sync: i use the official sync service so i can use obsidian on my several different devices at the same time without any hiccups... it's the only method i've tried that doesn't heck up when i open a note in two devices at the same time
- Git: ...buuut i also use the git plugin to back up my vault to github automatically
- Journals: better version of daily notes basically


### VS Code
- the code editor that everyone talks about ([VS Code website](https://code.visualstudio.com/))
- free and open source

this is where i make my websites ✨

#### VS Code plugins
- CSS Navigation: nobody talks about this plugin but it makes life sooo much easier??
- Code Spell Checker: for when i'm writing blogs n stuff in here
- Snippets Ranger

### Git
- annnd the version manager that everyone talks about ([Git website](https://git-scm.com/))
- free and open source

can't mention coding software without mentioning git! i'd be lost without it... in fact, i use it to track changes to my obsidian vaults and writing projects too, not just my code! it has saved my ass so many times when i do dumb stuff.

### Krita
- raster drawing app ([Krita website](https://krita.org/))
- free and open source 

#1 fav drawing software!!! i looove krita. it feels so good to draw in!!

### Inkscape
- vector design app ([Inkscape website](https://inkscape.org/))

### Affinity Photo/Affinity Designer
- "professional" graphic design apps 
- apparently they're not available anymore :c but the point more or less was that they were sorta close to Photoshop/InDesign capabilities but they were a one-time purchase instead of a subscription
- tbh i don't use them often, only when there's something i can't do in krita or inkscape
